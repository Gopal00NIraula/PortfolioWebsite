Guess Who Game with Figma

Project Overview

“Guess Who?” is a digital redesign of the classic character-guessing board game, adapted for web and mobile platforms. The goal of this UX/UI project was to create a visually engaging, intuitive, and responsive interface for both casual players and nostalgic fans. The design focuses on interactive storytelling, accessible layouts, and playful aesthetics to replicate the feel of the original game while enhancing it for the digital age.

Design Process

  • Research & Inspiration

    I began by analyzing classic board game UIs. Key references included card-based interfaces, avatar selection flows, and nostalgic color palettes that echo the physical version of the game.

  • Wireframing

    Created low-fidelity wireframes to map out the game play flow.

  • Visual Design

    Color was critical in setting a playful mood. I chose bold, saturated backgrounds, simple iconography, and animated character cards to draw the player in.Typography was kept simple and bold (e.g., Poppins / Montserrat) to maintain readability across devices. Shadows and hover states add subtle feedback without overwhelming the user. Using Figma's component-based design, I created reusable character cards.

What I learned

  • Designing for fun and function simultaneously.
  • Balancing visual storytelling with usability.
  • Using Figma's auto layout and variant tools efficiently.
  • Thinking from a user-first gameplay perspective rather than just visual design
