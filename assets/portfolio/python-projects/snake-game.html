Snake Game with Tkinter

Project Overview

This project is a modern recreation of the classic Snake arcade game built using Python's Tkinter library for GUI rendering. It demonstrates fundamental game development logic such as collision detection, keyboard interaction, dynamic rendering, and object-oriented programming. The game loop runs smoothly using after() timing, and the user controls the snake using arrow keys or WASD, collecting food to grow the snake and score points.

Features

  • Real-time Snake Movement

    Continuous gameplay using Tkinter's after() function to simulate frames.

  • Keyboard Controls

    Directional movement using Arrow keys and WASD keys.

  • Score Tracking

    Dynamic scoreboard that updates with each food collected.

  • Collision Detection

    Game ends when the snake collides with the wall or itself.

  • Random Food Placement

    Food appears at random coordinates in a grid-aligned fashion.

  • Clean UI Design

    Monochromatic snake color with vivid green food on a black background for contrast.
